UI ページ

初級 アーティスト デザイナー

UI ページ（UI Page） は、ボタンやグリッドなどの UI 要素の集まりです。 メニュー画面やタイトル画面などのように、ゲーム中の UI パーツを構成します。

構成的には、UI ページはシーンエディターで言うシーンに相当し、UI 要素はシーンで言うエンティティに相当します。要素もまた、エンティティのように、親と子を持つことができます。

各 UI シーンは、UI エディターのタブとして個別に開かれます。UI ページの編集方法については、UI エディターを参照してください。

UI ページを作成する

アセットビューで、[Add asset] > [UI] > [UI page] をクリックします。

すると、Game Studio は、アセットビューに UI ページを追加します。

UI 要素を使って UI ページを作成する

  1. UI エディターで、1 つまたは複数の要素を選択します。

  2. 右クリックして、[Create page from selection] を選択します。

Game Studio は、選択された要素を持ったページを生成します。

UI ページを開く

アセットビューで、 [UI page] アセットをダブルクリックします。

すると、UI ページが UI エディターで開かれます。

UI ページに UI 要素を追加する

グリッドやボタンのような要素を追加するには、それを UI ライブラリ から UI ページまたはビジュアルツリーにドラッグします。

UI エディターの使い方の詳細については、UI エディターを参照してください。
