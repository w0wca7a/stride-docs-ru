Interface IPlayableSound

Namespace
Stride.Audio
Assembly
Stride.Audio.dll

Interface for a playable sound. A playable sound can loop (ref IsLooping), be played (ref Play()), be paused (ref Pause()), be resumed (ref Play()), be stopped (ref Stop()) and be attenuated (ref Volume). To query the current state of a sound use the PlayState property. To stop a sound after its currently loop set IsLooping = false;

The global volume at which the sound is played.

Remarks

Volume is ranging from 0.0f (silence) to 1.0f (full volume). Values beyond those limits are clamped.

Methods

Remarks

Remarks
